# 📖 OOP school library <a name="about-project"></a>

**OOP School Library** implements Ruby classes for `Person`, `Student`, and `Teacher` along with other components relevant to a School Library system, adhering to provided requirements.

### Key Features <a name="key-features"></a>

- **Add new books.**
- **Add new students or teachers.**
- **Save records of who borrowed a given book and when.**
